# CoachFit MVP

Mobile-first web-app для персональных тренировок. MVP работает без бэкенда и сохраняет данные в `localStorage`, чтобы сценарий можно было сразу проверить в браузере. Слой данных назван так же, как будущие таблицы Supabase/Firebase.

## Запуск

```bash
cd /Users/sergeychirkov/fitness-coach-mvp
python3 -m http.server 5174
```

Открыть: `http://localhost:5174`

## Что входит в MVP

- Профиль пользователя: имя, возраст, рост, вес, цель, уровень, ограничения, частота.
- Программа тренировок по 5-дневному циклу: грудь + трицепс, спина + бицепс, ноги, плечи, руки.
- Экран тренировки: фактический вес, фактические повторы, выполнено, RPE, комментарий, добавление/удаление подходов, пропуск упражнения.
- История тренировок: даты, объём, максимальный рабочий вес, пропущенные упражнения, динамика силы и объёма.
- База упражнений: группа, тип, оборудование, техника, ошибки, замены.
- Админка программы: добавление дней, добавление упражнения, редактирование подходов, повторов и веса, копирование шаблона недели.
- Автоматическая рекомендация после завершения тренировки.
- Генетический профиль: правила AI Coach, диапазон 8-15 повторений, плавная прогрессия, Zone 2, кофеин, контроль талии/аппетита и защита связок.

## Текущая программа

1. День 1 — Грудь + Трицепс: жим штанги лёжа, жим гантелей на наклонной, сведение рук, трицепс на блоке, разгибание из-за головы, французский жим.
2. День 2 — Спина + Бицепс: верхний блок, горизонтальная тяга, тяга гантели одной рукой, подъем штанги на бицепс, молотки, скамья Скотта.
3. День 3 — Ноги: гакк-присед, жим ногами, разгибание ног, сгибание ног лёжа, сгибание ног сидя, икры.
4. День 4 — Плечи: жим гантелей сидя, подъемы в стороны, обратная разводка, подъем перед собой, шраги.
5. День 5 — Руки: разгибание на блоке, разгибание из-за головы, подъем EZ-штанги, молотки, скамья Скотта, отжимания на брусьях/узким хватом.

Стартовые веса оставлены пустыми как `вес вписать`: пользователь вводит фактический рабочий вес на первой тренировке, после чего рекомендации строятся от него.

## Логика тренера

- План выполнен и RPE ниже 8: вес увеличивается примерно на 3.5% с округлением до 2.5 кг.
- План выполнен и RPE 8-9: вес сохраняется.
- План не выполнен и RPE 9-10: вес снижается примерно на 4%.
- План не выполнен при RPE ниже 9: уменьшаются повторы.
- Два пропуска подряд: предлагается замена.
- Три одинаковых результата подряд: предлагается изменить схему подходов/повторов.
- Частые RPE 9-10: предлагается разгрузочная тренировка.

## План развития

1. Supabase/Firebase: авторизация, Postgres/Firestore, правила доступа, синхронизация между устройствами.
2. Графики: полноценные линии силы, объёма, RPE, тоннажа по мышечным группам.
3. ИИ-тренер: объяснение рекомендаций, адаптация программы под сон, самочувствие и историю.
4. Видео-анализ техники: загрузка подхода, чек-лист ошибок, подсказки по амплитуде и темпу.
5. Apple Health: вес, пульс, активность, сон, восстановление.
6. Экспорт PDF/Excel: отчёт клиента, прогресс по упражнениям, рекомендации тренера.
7. Кабинет тренера: доступ к клиентам, комментарии, корректировка программы, шаблоны.
